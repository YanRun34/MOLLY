# MOLLY 20周年沉浸式互动 H5

基于 React + Vite + Tailwind CSS + Framer Motion 的多页互动体验。

## 一键预览（推荐）

双击项目根目录的 **`start.bat`**，浏览器访问：

**http://127.0.0.1:8080**

`dist/` 已是可部署的静态成品；`start.bat` 会启动本地静态服务器（不要用 `file://` 直接打开 HTML）。

## 开发模式

```bash
npm install
npm run dev
```

## 构建部署

```bash
npm run build
```

将 `dist/` 整个目录上传到任意静态托管（Nginx、OSS、GitHub Pages 等）。

## 背景音乐与音效

- 点击「开启旅程」后解锁音频（符合浏览器自动播放策略）
- 全程轻柔合成背景音乐循环播放
- 点击、收集、过关、完成等操作配有音效
- 右上角按钮可随时静音/开声

音效文件位于 `public/assets/audio/`，可替换为自有素材。

## 专属人格海报

- 11 张成长图位于 `public/assets/worlds/`
- 「记忆宇宙」卡片与详情弹层展示对应图片
- 生成海报时：**浏览最多的旅程**优先；未浏览则**随机**一张，并配上专属文案
- 浏览器本地 Canvas 合成，无需第三方 API

## 页面流程

1. **首页 KV** — 全屏海报，点击下方「开启旅程」区域进入
2. **上色小游戏** — 点击三色块找回颜色（3/3 后自动进入下一页）
3. **记忆宇宙** — 11 个 MOLLY 世界卡片，各含 1 个专属小游戏（完成后显示「已完成」）
4. **长按拥抱** — 长按 MOLLY 圆钮填满进度条
5. **结果页** — 人格结果与 CTA
